# GenTable

`genTable` creates a table widget on screen that can be used to display the contents of a function table or the contents of a file. When `selectableRegions` is enabled, users can interactively select portions of the waveform by clicking and dragging. The start and end points can be picked up using the 'start' and 'end' channels. Note this widget is non-automatable. Therefore, its channels will not appear in a DAW as parameters.
