# Form

Form creates the main application window. pluginId() is the only required identifier. The default values for size are 600x300. Form identifiers are mostly static apart from `latency()` which can be updated dynamically. 

## Syntax
```json
{
  "type": "form",
  "bounds": "12, 20, 352, 228",
  "channel": "infoText",
  "caption": "string",
  "size": {
    "width": 100,
    "height": 100
  },
  "bundle": [
    "./folder1",
    "file2",
    "etc.."
  ],
  "pluginId": "plug",
  "guiRefresh": 1,
  "guiMode": "val",
  "import": [
    "file1",
    "file2",
    "etc.."
  ],
  "titleBarColour": "colour",
  "style": "style",
  "typeface": "fontFile",
  "latency": 32,
  "opcodeDir": "dir",
  "colour": "colour",
  "fontColour": "colour"
}
```
