```json
"style": {
  "opacity": 1,
  "borderRadius": 2,
  "borderWidth": 1,
  "borderColor": "#222222",
  "fill": "#0295cf"
},
```

- Colors accept named CSS colors, hex (e.g., #RRGGBB or #RRGGBBAA), or rgb/rgba strings.
- Label typography is configured under `label.*`.
- The dropdown inherits border and fill colors from the style object.
