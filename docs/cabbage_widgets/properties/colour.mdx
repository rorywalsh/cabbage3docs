```json
"colour": "colour"
"colour":{ "on":"colour", "off":"colour" } // button and checkbox
"colour":{ "fill":"colour", "background":"colour" }  // genTable
```

This property sets the main color, which can be any valid CSS or HTML color string. It also accepts hexadecimal, RGB or RGBA values, where each channel ranges from 0 to 255. For example, `"colour":[0, 0, 255]` creates blue, while `"colour":"#ff0000"` generates a red colour.

For widgets with two states, like buttons, you can pass a JSON object with "on" and "off" keys, each containing a color string. `gentable` uses `fill` and `background` colour properties.
