```json
"style": {
  "track": {
    "width": "auto"
  }
}
```

Sets the width of the horizontal/vertical slider track. Use a number (pixels) or `"auto"` to have it scale with the widget size.

- Type: number | "auto"
- Default: "auto"
- Applies to: horizontalSlider, verticalSlider (and any others that expose `style.track.width`)
