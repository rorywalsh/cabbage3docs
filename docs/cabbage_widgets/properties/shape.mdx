```json
"shape": {
    "borderRadius": 6,
    "borderWidth": 0,
    "borderColor": "#dddddd",
    "fill": "#0295cf",
    "background": "#000000"
}
```

Shape defines the visual appearance of the widget's background and borders. borderRadius sets corner rounding, borderWidth and borderColor control the border, fill sets the background color, and background is used for secondary backgrounds in some widgets.
