```json
"state": {
    "on": {
        "backgroundColor": "#3d800a",
        "textColor": "#dddddd"
    },
    "off": {
        "backgroundColor": "#3d800a",
        "textColor": "#dddddd"
    },
    "hover": {
        "backgroundColor": "#4ca10c"
    },
    "active": {
        "backgroundColor": "#2d6008"
    }
}
```

State defines the colors for different button states: on, off, hover, and active. Each state can have backgroundColor and textColor properties.
