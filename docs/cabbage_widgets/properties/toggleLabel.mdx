```json
"label": {
  "text": {
    "off": "Start",
    "on": "Stop"
  },
  "align": "center"
}
```

Configures the label shown inside widgets that switch between off/on states, such as `button` and `fileButton`.

- text.off: string — the text shown when the widget value is `0`.
- text.on: string — the text shown when the widget value is `1`.
- align: string — optional text alignment for widgets that support it, such as `button`.

Typography and colors are configured via `style.*`, including `style.on.textColor`, `style.off.textColor`, and related state styles.
