```json
"fontColour": {"on":[255, 0, 0], "off":"#ff0000"}
```

Sets the font colour. Any CSS or HTML colour string can be passed to this identifier. The font colour identifier can also be passed an RBG, or RGBA array. All RGB channel values must be between 0 and 255. In the above example, the "on" colour is set as an RGB array, while the "off" colour is set as a hexadecimal string.
