```json
"colour": {
    "whiteNote": "#ffffff",
    "arrowBackground": "#0295cf",
    "keydown": "#93d200",
    "blackNote": "#000000"
}
```

Colour properties can be any valid CSS or HTML color string. They can also accepts be hexadecimal strings, or RGB / RGBA array, where each channel ranges from 0 to 255. For example, `"colour":{"whiteNote":[0, 0, 255]}` or `"colour":{"keydown":""#ff0000"}`. `fill` sets the table colour, while `stroke.outline` sets the outline colour.
