```json
"colour": {
    "on": {
        "fill": "#0295cf",
        "stroke": {
            "colour": "#dddddd",
            "width": 2
        }
    },
    "off": {
        "fill": "#a8d388",
        "stroke": {
            "colour": "#dddddd",
            "width": 2
        }
    }
}
```

This property sets the main colour of a button. Colour properties themselves can be any valid CSS or HTML color string. They can also accepts be hexadecimal strings, or RGB / RGBA array, where each channel ranges from 0 to 255. For example, `"colour":{"on":{"fill":[0, 0, 255]}}` or `"colour":{"on":{"fill":""#ff0000"}}`.

The `stroke` object set the outline colour and buttons.
