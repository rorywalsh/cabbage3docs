```json
"style": {
  "track": {
    "thickness": "auto"
  }
}
```

Sets the thickness of the rotary slider track. Use a number (pixels) or `"auto"` to have it scale with the widget size.

- Type: number | "auto"
- Default: "auto"
- Applies to: rotarySlider (and other sliders that expose `style.track.thickness`)
