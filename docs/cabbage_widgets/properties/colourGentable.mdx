```json
"colour": {
    "fill": "#a8d388",
    "stroke": {
        "colour": "#dddddd",
        "width": 1
    }
}
```

Colour properties can be any valid CSS or HTML color string. They can also accepts be hexadecimal strings, or RGB / RGBA array, where each channel ranges from 0 to 255. For example, `"colour":{"fill":[0, 0, 255]}` or `"colour":{"fill":""#ff0000"}`. `fill` sets the table colour, while `stroke.outline` sets the outline colour.
