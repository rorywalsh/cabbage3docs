```json
{
  "style": {
    "opacity": 1,
    "borderRadius": 4,
    "fill": "#0295cf"
  },
  "label": {
    "text": "",
    "offsetY": 0,
    "fontFamily": "Verdana",
    "fontSize": "auto",
    "color": "#dddddd",
    "textAlign": "center"
  },
  "valuePrefix": "",
  "valuePostfix": ""
}
```

Notes
- fill controls the background colour of the slider block.
- label.fontSize: "auto" scales with height; set a number (px) to fix it.
- label.textAlign: left | center | right.
- Colours accept CSS strings or RGB arrays [r,g,b].
